OneStream Dashboard Wireframe Template
==================================================

This template helps you design OneStream dashboards.

Files:
  1_Instructions.csv - Usage instructions
  2_Basic_Info.csv - Workspace settings
  3_Parameters.csv - Parameter definitions
  4_Components.csv - Component definitions
  5_Dashboard_Layout.csv - Dashboard layout structure
  6_Component_Positioning.csv - Component placement
  7_Grid_Visual_Guide.csv - Layout visualization

To use:
1. Open these CSV files in Excel (or import into one workbook)
2. Fill in your dashboard design
3. Save the workbook as .xlsx
4. Run: python scripts/convert_excel_to_dashboard.py your_file.xlsx
